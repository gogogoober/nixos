# Claude Code baseline

This file is the user-level `~/.claude/CLAUDE.md` shipped from the NixOS
config at `assets/claude/CLAUDE.md`. It is overwritten on every
`nixos-rebuild` / `home-manager switch` by the `claude-skills-seed`
activation hook.

Edit the copy in this repo, not `~/.claude/CLAUDE.md` directly — local edits
are discarded on the next rebuild.
